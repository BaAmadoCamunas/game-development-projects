# Tiny Tussle

*Tiny Tussle* is a 2D local multiplayer platform shooter developed as part of my **Bachelor's Degree in Video Game Design and Development** at Universidad Rey Juan Carlos.

Players compete to collect gems while using randomly spawned weapons and unique character abilities to defeat their opponents. The player with the highest score when the match ends wins.

## Overview

Tiny Tussle combines fast-paced platforming with competitive multiplayer gameplay. Players must collect gems while fighting opponents using different weapons and character-specific abilities across multiple arenas.

The project was designed for browser play and focuses on quick, accessible multiplayer matches.

---

## Gameplay Features

- Local multiplayer for up to 2 players
- Competitive gem collection system
- Random weapon spawning
- Character-specific special abilities
- Multiple playable arenas
- Score-based victory system
- Pixel art visual style
